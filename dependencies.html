DependenciesÂ¶

ionEngine depends on the following external libraries:

  • Header Only
    • stb - Image loading/writing, integer divide methods
    • tinyformat - Typesafe C++11 printf-style string formatting
    • GLM - Vector and matrix math
    • imgui - GUI components
    • Catch - Unit testing (only necessary to build/run unit tests)
    • GLAD - OpenGL extension loading
    • dirent - Directory navigation - Only on windows (UNIX-variants have it by default)
  • Binary
    • GLFW - Windowing and other OS functions

For instructions on how to build and install, see the installation page.
