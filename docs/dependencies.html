Dependencies

ionEngine depends on the following external libraries:

Header Only

  • stb
    • Image loading/writing, integer divide methods
    • Latest commit as of 5/11/2017: c711058
  • tinyformat
    • Typesafe C++11 printf-style string formatting
    • v2.1.0
  • GLM
    • Vector and matrix math
    • v0.9.8.4
  • imgui
    • GUI components
    • Latest commit as of 5/11/2017: df52f46
  • Catch
    • Unit testing (only necessary to build/run unit tests)
    • v1.9.3
  • GLAD
    • OpenGL extension loading
  • tinyobjloader
    • Import .obj meshes
    • v0.9.25
  • simpleini
    • Read config from .ini files
    • Latest commit as of 5/11/2017: 03e27b2
  • dirent
    • Directory navigation
    • (Windows Only)
    • Latest commit as of 5/11/2017: 8b1db50

Binary

  • GLFW
    • Windowing and other OS functions
    • v3.2.1

For instructions on how to build and install, see the installation page.
